# AI Learning Buddy – KL Vyshnavi

## 📖 Overview

AI Learning Buddy – KL Vyshnavi is an AI-powered educational web application built using **Streamlit** and **Google Gemini AI**. It helps students learn Data Science concepts through simple explanations, real-life examples, quizzes, and interactive question-answer sessions.

The application is designed to make learning engaging, easy to understand, and suitable for beginners.

---

## ✨ Features

- 📚 Explain Data Science concepts in simple language
- 💡 Generate real-life examples
- 📝 Create multiple-choice quizzes with answers
- 💬 Ask AI any Data Science-related question
- 🎯 Beginner-friendly interface

---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/kancharla-vyshnavi/AI-Learning-Buddy-KL-Vyshnavi.git
```

2. Navigate to the project folder

```bash
cd AI-Learning-Buddy-KL-Vyshnavi
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Add your Gemini API Key.

5. Run the application

```bash
streamlit run app.py
```
